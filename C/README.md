# C Language Version / Gomoku(*오목*)
By gnyontu39@gmail.com

### INFORATION
* Windows operating system only.
  * Test OS : Windows XP/7/10
* 2 Player Game
  * It does not support game with computer.
* You can control the size of the game's map.
  
### HOW TO PLAY?
* Move
  * W,A,S,D
* Check
  * Space
  
### MAP
* ● : 1Player
* ○ : 2Player
* ※ : Pointer
